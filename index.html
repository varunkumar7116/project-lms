HTML Beginner Tags With Visuals

<title> Tag

Explanation: The <title> tag defines the title of the HTML document. It appears in the browser tab, bookmarks, and search engine results. Each page should have a unique and meaningful title for better user experience and SEO.

Syntax:

<head>
  <title>Page Title</title>
</head>

Example:

<!DOCTYPE html>
<html>
<head>
  <title>My Portfolio</title>
</head>
<body>
  <h1>Welcome to My Portfolio</h1>
</body>
</html>
<p> Tag

Explanation: The <p> tag defines a paragraph.

Syntax:

<p>Your text here</p>

Example:

<p>This is a simple paragraph.</p>
<button> Tag

Explanation: The <button> tag defines a clickable button on the webpage.

Syntax:

<button>Click Me</button>

Example:

<button onclick="alert('Hello!')">
  Click Me
</button>
<ul> Tag

Explanation: The <ul> tag creates a bullet-point list.

Syntax:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Example:

<ul>
  <li>Milk</li>
  <li>Eggs</li>
</ul>
<img> Tag

Explanation: The <img> tag displays an image on the page.

Syntax:

<img src="image.jpg" alt="Image Description">

Example:

<img src="cat.jpg" alt="Cute Cat">
